# antolick.ai

Personal portfolio website serving as an e-resume and project showcase, hosted at [antolick.ai](https://antolick.ai).

## Tech Stack

- **Framework:** Next.js 16 (React 19) with TypeScript
- **Styling:** Tailwind CSS 4
- **Deployment:** Docker (static export)

## Features

- **Projects** — Highlighted portfolio of professional and personal work
- **About** — Background, education, current role, and homelab details
- **Space Theme** — Animated starfield background, constellation dividers, shooting star effects, and hyperspace warp easter egg

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the site.

## Build & Deploy

```bash
# Static export
npm run build

# Docker
docker compose up --build
```
